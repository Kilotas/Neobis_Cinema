# Cinema Project

**Описание:** Это проект на Django Rest Framework (DRF) для управления системой кинотеатра. Он включает в себя функционал по управлению пользователями, залами, фильмами, заказами и многим другим.

### Установка

1. **Клонирование репозитория:**

git clone https://github.com/Kilotas/Neobis_Cinema.git

2. **Переход в директорию проекта:**

cd NeobisCinema


3. **Установка зависимостей:**

pip install -r requirments.txt


### Конфигурация

1. **Настройка конфигурации базы данных:**
- Настройте параметры базы данных в файле `settings.py` в соответствии с вашими предпочтениями.

2. **Применение миграций базы данных:**

Python manage.py migrate


2. **Доступ к интерфейсу администратора Django:**
- Перейдите по адресу `/admin` для управления пользователями, фильмами, залами и т. д.

3. **Исследование конечных точек API:**
- Пользователи: `/users/`
- Залы: `/hall/`
- Фильмы: `/movies/`
- Заказы: `/orders/`

### Документация API

Вы можете получить доступ к документации API и изучить доступные конечные точки, используя следующие ссылки:

- Схема API: `/api/schema`
- Документация API (Swagger UI): `/api/docs`
